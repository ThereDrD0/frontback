# Практическая работа 18

Подготовка к контрольной работе №3.

В контрольную работу входят практики `pr13`-`pr17`:

- `pr13` - список дел с `Service Worker` и офлайн-режимом;
- `pr14` - добавлен `Web App Manifest`;
- `pr15` - добавлены `HTTPS` и `App Shell`;
- `pr16` - добавлены `WebSocket` и `push`-уведомления;
- `pr17` - добавлены напоминания и откладывание уведомлений.

## Проверка

Основное приложение для контрольной запускается из `pr17`:

```bash
cd ../pr17/client
npm install
npm start
```

Адрес: `http://localhost:3001`

После запуска нужно открыть приложение, включить уведомления, создать обычную заметку и заметку с напоминанием на будущее время.
