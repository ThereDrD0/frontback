# Практическая работа 11

Сделаны роли `user`, `seller`, `admin`, маршруты для пользователей и React-клиент с разграничением действий.

## Запуск

Сервер:

```bash
cd server
npm install
npm start
```

Клиент:

```bash
cd client
npm install
npm start
```

Сервер: `http://localhost:3000`

Клиент: `http://localhost:3001`

Тестовый администратор: `admin@example.com` / `admin123`
