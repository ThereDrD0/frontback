# Практическая работа 10

Сделан сервер с refresh-токенами и минимальный клиент на React с хранением токенов в `localStorage`.

## Запуск

Сервер:

```bash
cd server
npm install
npm start
```

Клиент:

```bash
cd client
npm install
npm start
```

Сервер: `http://localhost:3000`

Клиент: `http://localhost:3001`
