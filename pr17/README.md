# Практическая работа 17

Приложение для заметок с `WebSocket`, `push`-уведомлениями и напоминаниями.

Что сделано:

- добавление обычной заметки;
- добавление заметки с датой и временем напоминания;
- хранение заметок в `localStorage` с `id` и `reminder`;
- отправка запланированного `push`-уведомления с сервера;
- кнопка в уведомлении для откладывания напоминания на 5 минут.

## Запуск

```bash
cd client
npm install
npm start
```

Адрес: `http://localhost:3001`
