# Практическая работа 12

Подготовлена контрольная работа №2 по результатам практик `pr7` - `pr11`.

Сделано:

- регистрация и вход пользователя
- JWT access и refresh токены
- роли `user`, `seller`, `admin`
- защищенные маршруты
- работа с товарами
- управление пользователями для администратора

## Запуск

Сервер:

```bash
cd server
npm install
npm start
```

Клиент:

```bash
cd client
npm install
npm start
```

Сервер: `http://localhost:3000`

Клиент: `http://localhost:3001`

Тестовый администратор: `admin@example.com` / `admin123`
